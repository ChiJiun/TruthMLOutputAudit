# Week 19 Breast Cancer Repeatability

This folder repeats the core FL and DP baseline experiments on a similar tabular binary classification dataset:

- `sklearn.datasets.load_breast_cancer`

The goal is to check whether the qualitative behavior observed on Adult Income also appears on another small structured dataset.

## Run

```powershell
python week19_breast_cancer_repeatability\run_breast_cancer_repeats.py
```

## Outputs

- `results/repeat_runs.csv`
- `results/summary.md`
- `results/config.json`
